# Useroam Teknoloji Homebrew Tap

Official Homebrew tap for [Noden](https://noden.useroamteknoloji.com/) and other Useroam Teknoloji macOS applications.

## Install Noden

```bash
brew install --cask useroamteknoloji/tap/noden
```

Or tap the repository first:

```bash
brew tap useroamteknoloji/tap
brew install --cask noden
```

## Update

```bash
brew upgrade --cask noden
```

## Uninstall

```bash
brew uninstall --cask noden
```

Noden supports macOS 13 Ventura and later on Apple Silicon and Intel Macs.
